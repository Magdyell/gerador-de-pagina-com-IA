# 🚀 Gerador de Páginas com IA

Um gerador de páginas web alimentado por Inteligência Artificial, onde o usuário descreve a página desejada em linguagem natural e a IA gera automaticamente o código HTML e exibe o resultado em tempo real.

## ✨ Funcionalidades

- 🎨 Interface moderna e responsiva
- 🤖 Geração de páginas através de prompts
- 📄 Exibição do HTML gerado
- 👀 Pré-visualização em tempo real utilizando iframe
- ⚡ Integração com APIs de IA (Gemini/OpenAI)
- 🌙 Tema escuro moderno

---

## 📸 Demonstração

Digite um prompt como:

```text
Crie uma landing page moderna para uma empresa de tecnologia.
```

A aplicação envia o pedido para a IA e exibe:

- Código HTML gerado
- Pré-visualização da página

## ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/gerador-ia.git
```

Entre na pasta:

```bash
cd gerador-ia
```

Instale as dependências:

```bash
npm install
```

---

## 🔑 Configuração da API

Crie um arquivo `.env` na raiz do projeto:

```env
GEMINI_API_KEY=SUA_CHAVE_AQUI
```

Você pode obter uma chave em:

https://aistudio.google.com/app/apikey

---

## ▶️ Executando o Projeto

Inicie o servidor:

```bash
node server.js
```

Abra no navegador:

```text
http://localhost:3000/pagina.html
```
